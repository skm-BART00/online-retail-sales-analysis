\# Online Retail Sales Analysis 📊



This project analyzes transactional sales data from an online retail company using Python and Power BI.



The goal of the analysis was to explore sales performance, identify revenue trends, and build an interactive dashboard.



\## Tools Used



\- Python

\- Pandas

\- Matplotlib

\- Jupyter Notebook

\- Power BI



\## Dataset



UCI Online Retail Dataset containing transactional sales data from 2010–2011.



\## Analysis Process



1\. Data cleaning and preprocessing

2\. Exploratory data analysis

3\. Revenue trend analysis

4\. Product performance analysis

5\. Geographic sales analysis

6\. Dashboard creation in Power BI



\## Key Insights



\- Total revenue exceeded \*\*10.6M\*\*

\- Sales peak occurs in \*\*November\*\*, indicating strong seasonal demand

\- Over \*\*20k orders\*\* were processed

\- The company operates across \*\*38 countries\*\*
